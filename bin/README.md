# login-system-1
### Requirements
- JWT for Auth
- Bachekd -> Spring Booot
- Frontend -> React Js

### Goal
- When
  - Username: admin, Password: admin -> http://localhost:3000/admin
  - Username: user1, Password: user1 -> http://localhost:3000/user

- UI
  - User -> "Hello User"
  - Admin -> "Hello Admin"

- React's File Structure
  - src
    - components (element)
    - pages (layout)

- Spring Boot
  - src
    - controller (API)
    - model
    - auth (JWT)
    - security
  - webapp (frontend)
